# HRMS - Human Resource Management System 🚀

A modern, scalable backend API for managing organizational resources, employees, and payroll operations.

## 🛠 Technologies

- **Runtime:** [Node.js](https://nodejs.org)
- **Framework:** [Express.js](https://expressjs.com)
- **Database:** [MongoDB](https://www.mongodb.com) with [Mongoose](https://mongoosejs.com)
- **Security:** [JWT](https://jwt.io) (JSON Web Tokens) & [Bcrypt.js](https://www.npmjs.com)
- **Environment:** [Dotenv](https://www.npmjs.com)

## ✨ Core Features

- **Secure Authentication:** JWT-based login with password hashing.
- **Role-Based Access (RBAC):** Distinct permissions for Admin, Manager, and Employee roles.
- **Employee Directory:** Complete CRUD for managing staff profiles.
- **Attendance System:** Track daily check-ins/check-outs.
- **Leave Management:** workflow for applying and approving leave requests.
- **Payroll Processing:** Automated salary calculations and status updates.

## 🚀 Getting Started

### Prerequisites

- Node.js installed
- MongoDB (Atlas or Local)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/shouvik8017/hrms-backend.git
   cd hrms-backend
   ```
2. **Install all packages**
   ```bash
   pnpm i
   ```
3. **Start the server**
   ```bash
   pnpm start
   ```
